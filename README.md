# MakeGUIButton
A Make button + console to have am easier control of your build


* First get GCC : https://gcc.gnu.org/install/download.html
* Second download the file (main.c)
* Third open your CMD go to the main.c path
* Fourth execute this command : gcc -o MakeGUI.exe main.c -mwindows
* Fifth open MakeGUI.exe and enjoy :)
